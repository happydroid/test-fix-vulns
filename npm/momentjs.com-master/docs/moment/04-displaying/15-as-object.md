---
title: As Object
version: 2.10.5
signature: |
  moment().toObject();
---


This returns an object containing year, month, day-of-month, hour, minute,
seconds, milliseconds.

```javascript
moment().toObject()  // {
                     //     years: 2015
                     //     months: 6
                     //     date: 26,
                     //     hours: 1,
                     //     minutes: 53,
                     //     seconds: 14,
                     //     milliseconds: 600
                     // }
```
